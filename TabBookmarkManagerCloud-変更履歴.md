# Tab Bookmark Manager Cloud 変更履歴

## 2026-06-02

### タイトル取得ボタンの配置を調整

- ブックマーク保存フォームの `タイトル取得` ボタンを、タイトル欄の横からURL欄の右端内側へ移動。
- URL入力中でもボタンが同じ行に収まるよう、URL入力欄の右側余白とスマホ幅でのボタンサイズを調整。

### URLからタイトルを取得する機能を追加

- ブックマーク保存フォームに `タイトル取得` ボタンを追加。
- URL入力後、タイトル欄が空の場合はURL欄の変更・フォーカスアウト時にもページタイトルを自動取得するようにした。
- 保存時にタイトル欄が空の場合も、ページタイトル取得を試してから保存するようにした。
- 直接取得できないサイト向けに、複数のCORSプロキシを順に試すフォールバックを追加。

### Cloud同期の安定性を改善

- Chrome拡張版と同じく、Supabaseへの全件upsertを50件ずつの分割処理に変更。
- 削除同期で全IDを `not in` 条件に詰め込む方式をやめ、Cloud側IDを取得して差分だけを分割DELETEする方式へ変更。
- ブックマーク件数が増えた場合のURL長超過・リクエスト肥大による同期失敗リスクを低減。
- ライトモードのタイトル色と `theme-color` をRSS Readerライトモードのロゴと同じ青系アクセント（`#64779f`）に変更。PC表示の `.full-title` にも明示適用。ダークモードは従来の赤系アクセントを維持。

## 2026-06-01

### Supabase 対応の単独 HTML/PWA 版を新規作成

- `index.html` を追加し、ブックマークの手入力保存、検索、編集、削除、Import / Export に対応。
- Supabase URL、anon key、共有IDを保存する Cloud 設定モーダルを追加。
- `tab_bookmarks` テーブル用の `supabase-schema.sql` を追加。
- `manifest.webmanifest` とホーム画面追加用アイコンを追加。
- Chrome 拡張版と同じ JSON データ形式を読み込めるようにした。

### HTML取扱説明書を追加

- `TabBookmarkManagerCloud-取扱説明書.html` を追加し、人間向けの操作手順とAI向けの実装メモを整理。
- Markdown版の取扱説明書はHTML版への案内に差し替え。

### カード操作ボタンの表示を調整

- ブックマークカード右上の編集・削除ボタンを、文字ボタンから `✎` と `×` のミニマルなアイコンボタンへ変更。
- カード内アクション専用の固定幅レイアウトを追加し、PC幅・スマホ幅のどちらでも横並びで揃うようにした。

### スマホ表示のヘッダーと保存フォームを調整

- スマホ表示ではヘッダータイトルを `Tab Bookmark Manager Cloud` から `TBMC` に短縮。
- スマホ表示ではヘッダーの Import / Export ボタンを非表示化。
- スマホ表示では「ブックマークを保存」パネルを初期非表示にし、画面下フッター左端の `B!` ボタンで開閉するようにした。

### スマホ表示の操作配置を再調整

- スマホ表示のヘッダータイトルを `TBMC` から `TBookmarkMC` に変更。
- 画面下のモバイルフッターを削除し、保存フォーム開閉用の `B!` ボタンをヘッダーの `Cloud` ボタン左隣へ移動。
- フッター削除に合わせてスマホ表示の下部余白を調整。
- ヘッダー内の `B!` ボタンの高さを `Cloud` ボタンと揃えるよう調整。
- 保存フォーム表示中も `B!` の文字が消えないよう、クリック後・フォーカス中の文字色と背景色を明示。
- スマホ表示のCloud設定画面が左上に寄らないよう、RSS Reader Cloud の設定画面と同じ中央配置・スマホ幅に調整。

### ファビコン・PWAアイコンを追加

- `icons/app-icon.svg` を追加し、Web Antenna 等と同じブルー系背景（`#64779f`）と白いブックマーク記号のデザインに統一。
- `favicon-16.png`、`favicon-32.png`、`apple-touch-icon.png`、`icon-192.png`、`icon-512.png` をSVG元データから生成。
- `index.html` に favicon 参照を追加し、`manifest.webmanifest` のテーマ色と背景色をブルーに変更。
